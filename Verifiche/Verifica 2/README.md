# Esercitazione 7d - Il Linguaggio C

# Obiettivi

Implementare tutte le funzioni nel file `menu.h`

## Contesto

Questo è un programma gestionale di una ristorante e si basa su tre file: `dishes.txt` e `menu.txt`.

Ogni piatto ha un nome e una descrizione.

Un menù consiste in una serie di piatti e prezzi (MenuItem)
